Train Scheduler

This is an app using Moment.js that allows a user to view train schedules. Moment.js calculates the time until the next train. User input is stored in Firebase.

Languages Used:

  • HTML
  • CSS
  • Bootstrap
  • Javascript
  • JQuery
  • Firebase
  • Moment.js
